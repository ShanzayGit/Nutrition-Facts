# Nutrition Label Project

## Overview

This project recreates a **Nutrition Facts label** using **HTML and CSS**. It focuses on practicing layout design, typography, spacing, borders, and styling techniques to build a realistic nutrition information panel.

## Features

* Structured semantic HTML layout
* CSS styling for accurate nutrition label appearance
* Flexible alignment using Flexbox
* Multiple divider styles for section separation
* Custom typography using Open Sans font
* Indentation for nested nutrition information
* Responsive centered card layout

## Technologies Used

* HTML5
* CSS3
* Google Fonts (Open Sans)
